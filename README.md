# 校园二手图书交易平台 - 微信小程序

这是一个校园二手图书交易平台的微信小程序项目，用于帮助学生之间进行二手图书的交易。

## 项目结构

```
├── app.js              // 小程序全局逻辑
├── app.json            // 小程序全局配置
├── app.wxss            // 小程序全局样式
├── project.config.json // 项目配置文件
├── pages/              // 页面文件夹
│   ├── index/          // 首页
│   ├── cart/           // 购物车
│   └── user/           // 个人中心
└── README.md           // 项目说明
```

## 功能特点

### 首页
- 顶部搜索栏
- 轮播图展示活动信息
- 学院分类导航
- 推荐图书展示
- 特价图书展示

### 购物车
- 商品列表展示
- 商品数量调整
- 商品删除
- 商品选择（单选/全选）
- 价格计算

### 个人中心
- 用户登录/登出
- 订单状态管理
- 个人功能菜单（我的图书、收藏、历史等）

## 如何运行

1. **准备工作**
   - 安装微信开发者工具
   - 注册微信小程序账号

2. **导入项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择本项目文件夹
   - 使用测试AppID（或输入您自己的AppID）

3. **运行项目**
   - 点击「编译」按钮
   - 在模拟器中查看效果
   - 或使用真机调试

4. **注意事项**
   - 当前为演示版本，使用模拟数据
   - 图片资源使用picsum.photos提供的示例图片
   - 支付等功能为模拟实现

## 开发说明

- 使用微信小程序原生开发方式
- 采用组件化思想设计页面结构
- 使用Flex布局进行页面排版
- 数据存储使用本地缓存模拟
