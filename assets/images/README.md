# 图片资源目录

此目录用于存放小程序中使用的图片资源。

## 注意事项

1. 请确保所有图片资源都使用合适的尺寸和格式
2. 建议使用压缩后的图片以提高加载速度
3. 对于图标类图片，建议使用SVG格式以保证清晰度

## 已使用的图片路径

- phone-icon.png - 电话图标
- email-icon.png - 邮箱图标
- time-icon.png - 时间图标
- add-image.png - 添加图片图标
- empty-order.png - 空订单状态图
- empty-notification.png - 空通知状态图

这些图片目前使用占位符，实际开发中请替换为真实图片资源。
